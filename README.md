# 🧪 Little Scientist's Lab

A cute and free chemistry toolkit designed for students, educators, researchers, and anyone working in the laboratory.

Little Scientist's Lab combines chemistry and web development to provide simple, reliable, and easy-to-use tools that make everyday laboratory calculations faster and more enjoyable.

🌐 **Website:**  🧪 [Little Scientist's Lab](https://mpinkzstudio.github.io/little-scientists-lab/)

---

## ✨ Features

### 🧪 Solution Chemistry
- Dilution Calculator
- Molarity Calculator
- Molecular Weight Calculator
- Solution Preparation Calculator

### ⚗️ Chemistry Analysis
- Strong Acid–Base Titration

### 📏 Laboratory Utilities
- Scientific Unit Converter

---

## 💡 Why I Built This

As an Applied Chemistry graduate, I often found myself using chemistry calculators that were outdated, cluttered, or difficult to use.

Little Scientist's Lab started as a personal project while I was learning web development. My goal was to combine my chemistry background with programming to build a clean, beginner-friendly toolkit that anyone can use for free.

I hope these tools can make studying, teaching, and laboratory work a little easier—and a little cuter. 🧪✨
